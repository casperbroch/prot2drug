This ReadMe file is only for the data processing scripts

papyrus_eda.py
This script was used to plot data and calculate stats for the report.

ESM_embeddings.py
This script was used the generate the ESM protein embeddings

make_ESM_embeddings_selection.py
This script was used to store a selection of the protein embeddings to a smaller file.

smiles_to_pathways.py
Was Used to generate the synthetic pathways of the molecules in Papyrus++.

combine_data.py
Was used to combine all the generated protein embeddings and pathways with Papyrus++.

train_val_test_split.py
Test many methods of splitting the data into train, validation and test sets with minimal contamination.
